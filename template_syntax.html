#纯HTML

双大括号会将数据解释为纯文本，而非 HTML 。为了输出真正的 HTML ，你需要使用 v-html 指令
被插入的内容都会被当做 HTML —— 数据绑定会被忽略。
注意，你不能使用 v-html 来复合局部模板，因为 Vue 不是基于字符串的模板引擎。
组件更适合担任 UI 重用与复合的基本单元。
